📖 Инструкция пользователя GCodeVA

GCodeVA — это визуализатор G-кода для 3D-печати, который позволяет загружать, просматривать, анализировать и экспортировать траектории печати в интерактивном 3D-пространстве прямо в браузере.

📷 [ВСТАВИТЬ СКРИНШОТ: Общий вид приложения с загруженной моделью]

1. Загрузка G-кода

Добавить файлы в приложение можно двумя способами:

  • Drag & Drop: Просто перетащите файлы с расширением .gcode, .nc или .ngc в центральную область экрана. Файл автоматически добавится в список и начнет строиться.
  • Через кнопку: В левой панели, в секции 「📁 Files」, нажмите кнопку Open. Выберите файлы в системном диалоге.
📷 [ВСТАВИТЬ СКРИНШОТ: Секция Files с открытым списком файлов и кнопкой Open]

В блоке файлов вы можете сортировать список по Имени, Размеру или Дате, нажимая на соответствующие вкладки. Выберите нужный файл кликом по нему в списке.

2. Построение и добавление моделей

  • Load Model (Replace) — Заменяет текущую модель в 3D-сцене на выбранную в списке. Старая модель удаляется.
  • ➕ Add Selected Model — Добавляет выбранную модель в сцену, не удаляя текущие. Новая модель автоматически сдвигается по оси X, чтобы не пересекаться с уже загруженными.

3. Управление камерой и видом

В секции 「🎥 Camera Views」 находятся предустановленные ракурсы:

  • Top / Bottom / Front / Back / Left / Right — стандартные ортогональные виды.
  • ISO 1–4 — изометрические виды с разных углов.
  • ↻ Rotate 360° — запускает плавный автоматический поворот модели. Нажмите еще раз, чтобы остановить.
💡 Ручное управление в 3D-окне: Вращение — левая кнопка мыши (ЛКМ), масштабирование — колесо мыши, перемещение — правая кнопка мыши (ПКМ).

Кнопка Grid / Axis в верхнем тулбаре скрывает и показывает сетку с осями координат (X — красная, Y — зеленая, Z — синяя).

4. Настройка параметров (Секция ⚙️ Parameters)

Здесь настраивается визуальное представление модели:

  • Tube Ø (mm): Диаметр трубки, из которой строится траектория экструзии. После изменения ползунка нажмите Apply Diameter.
  • Color: Выбор цвета пластика. Нажмите Apply Color, чтобы обновить вид.
  • Collinear °: Угол упрощения (оптимизации) траектории. Чем выше значение, тем больше точек удаляется. Это ускоряет рендер, но снижает детализацию на кривых. Нажмите Apply Collinear.
📷 [ВСТАВИТЬ СКРИНШОТ: Секция Parameters с ползунками и кнопками применения]

5. Материалы (Секция 💠 Materials)

Переключайте визуальный шейдер модели для лучшего восприятия:

  • Matte — матовый пластик (без бликов).
  • Plastic — глянцевый пластик (с бликами).
  • Steel — металлический материал (отражающий).
  • Glass — полупрозрачный стеклянный вид (удобно для просмотра внутренней структуры и ходов без экструзии).

6. Работа с несколькими моделями (Секция 📁 Manage Models)

Если вы добавили несколько моделей через кнопку Add, здесь появится список:

  • Выбор модели: Кликните по названию модели в списке, чтобы увидеть её габариты и позицию.
  • 👁 / 🚫 Видимость: Скрывает или показывает выбранную модель в 3D-сцене.
  • Position (mm): Вы можете вручную ввести координаты X, Y, Z для сдвига модели в пространстве и нажать Update, либо сбросить позицию кнопкой Reset.
  • 🗑 Remove — Удаляет модель из сцены.

7. Вкладка Analytics (Правая панель)

Включает статистику по первой загруженной модели (нажмите кнопку 📊 Calculate Analytics в секции Export, если данные не появились автоматически):

  • Dimensions: Габариты модели по осям X, Y, Z в миллиметрах.
  • Print Info: Количество точек экструзии, длина пути печати, максимальная скорость (мм/с), расчетное время печати, количество слоев и общий объем экструдированного пластика.
  • Optimization: Сравнение количества точек до и после оптимизации (удаления коллинеарных точек), а также процент сжатия.
📷 [ВСТАВИТЬ СКРИНШОТ: Вкладка Analytics с заполненными данными]

8. Вкладка Log (Правая панель)

Текстовый журнал всех действий приложения. Здесь отображаются:

  • Время загрузки и парсинга файлов.
  • Примененные параметры (цвет, диаметр, материал).
  • Ошибки (если файл не удалось прочитать).

9. Экспорт (Секция 💾 Export)

  • 📊 Calculate Analytics — Запускает расчет статистики для правой панели.
  • 🎥 Record 360° Video (WebM) — Записывает 3-секундное видео вращения модели на 360 градусов. Можно задать разрешение (W/H).
  • 📸 Save 10 View Photos — Делает 10 снимков модели с разных ракурсов (Top, Bottom, Front, Back, Left, Right, ISO 1-4) и скачивает их как PNG-файлы.

10. Интерфейс: Сворачивание панелей

В заголовке левой панели и в правой панели есть кнопки со стрелками ‹ и ›. Они сворачивают панели, чтобы освободить место для 3D-вида. Когда панель свернута, на краю экрана появляется кнопка для её возврата.
