Просмотр файлов GCode
Использует для построения коды G0 и G1

Возможные улучшения:
1. Производительность
Добавить прогресс-бар при загрузке больших G-code файлов

Оптимизировать рендеринг для очень больших моделей (миллионы точек)

Добавить кэширование загруженных файлов

2. Функциональность измерения
Измерение не только расстояния между точками, но и углов

Измерение длины выбранной линии (вдоль траектории)

Измерение площади или объема выделенной области

Сохранение результатов измерений в файл

3. Визуализация
Добавить цветовую индикацию скорости подачи (F-параметры)

Отображать разные типы перемещений разными цветами (G0 - быстрые, G1 - рабочие)

Добавить возможность скрывать/показывать отдельные слои (по Z)

Добавить легенду и цветовую шкалу

4. Анализ G-code
Определение границ модели (min/max координаты)

Расчет объема материала

Определение времени печати

Анализ экстремумов и проблемных зон

5. Экспорт
Экспорт модели в STL/OBJ/PLY форматы

Сохранение настроек вида в файл

Создание анимации облета модели

6. Интерфейс
Добавить строку статуса с информацией о модели

Добавить возможность изменять цвет фона

Добавить лупу для увеличения области

Добавить переключатель режимов рендеринга (wireframe/surface)

7. Работа со слоями
Разделение G-code на слои по высоте Z

Возможность показать только текущий слой

Анимация послойной печати

8. Импорт/Экспорт настроек
Сохранение настроек цвета, толщины, прозрачности

Сохранение позиции камеры

Экспорт текущего вида в изображение с настройками
