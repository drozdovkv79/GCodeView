Приложение GCode View and Analytics
Программа на Python 3.13 PyQt6 PyVista для macos для просмотра и анализа GCode (несколько миллионов точек)
Требования:
1. Вся работа программы идет в одном интерфейсе. Окно разбито на три части. Слева элементы управления, по цениру трехмерная модель, справа закладка "Аналитика", "Параметры" и "Лог" для вывода данных с аналитикой и логов консоли
2. Добавь кнопку выбрать директорию с файлами .gcode. При нажатии открывается диалог выбора директории и отображается список файлов .gcode из выбранной директории.
6. При нажатии на кнопку "Анализировать" читается выбранный в списке файла GCode и в закладку аналитика вводится: 1. Размер файла, 2. количество точек, размер модели высота, длина, ширина, 3. минимальное и максимальное расстояние между точками по оси  X, Y, Z отдельно 4. количество слоев, 5. минимальное и максимальное количество точек на слое, 6. количество материала на модель и минимальное и максимальное количество материала на каждый слой. 7. минимальное и максимальное количество материала на одну точку, 8.Расчет объема материала 9. Анализ экстремумов и проблемных зон
7. При нажатии на кнопку "Визуализировать" читается выбранный в списке файла GCode, проесс загрузки и построения подели отображается на прогресс баре, и отображается трехмерная визуализация модели в виде плавных непрерывных трубок с диаметором 4мм, разбитых послойно. Модель центрируется по оси X,Y по оси Z устанавливается в 0. Камера масшабируется что бы было видно всю модель.
8. На закладке "Параметры" добавить поле для ввода диаметра трубы и по нажатию кнопки "Обновить" обновляется визуализация модели с новым диаметром.
8. Используй для построения коды G0 и G1, показывай в модели только то что выдается с материалом, не пустой ход
8. На закладке "Настройка" сделай группу кнопок выбора цвета модели материала модели: Матовый (по умолчанию), Пластик, Гипс, Сталь, Стекло, При нажатии на кнопку меняется цвет модели в визуализации.
9. При нажатии на кнопку "Фото" создание и сохранение фотографий: спереди, сзади, слева, справа, сверху, снизу, ИСО фронт со всех 4 строн, ИСО зад со всех 4 строн
10. При нажатии на кнопку "Видео" создание анимации облета модели и сохранение в файл. во время записи видео парнель сначала должна встать в позицию ISO и сделать оборот вокруз вертикальной оси на 360 градусов. Дополнительно показывай прогресс создания видео и что бы программа не зависала
11. кнопка "Экспорт" для сохранения модели в формат STL, GLB или PLY форматы
12. Фон окна для модели сделай silver. 
13. Шрифт везде Arial 12pt


Просмотр файлов GCode
Использует для построения коды G0 и G1

Возможные улучшения:
1. Производительность
Добавить прогресс-бар при загрузке больших G-code файлов

Оптимизировать рендеринг для очень больших моделей (миллионы точек)

Добавить кэширование загруженных файлов

2. Функциональность измерения
Измерение не только расстояния между точками, но и углов

Измерение длины выбранной линии (вдоль траектории)

Измерение площади или объема выделенной области

Сохранение результатов измерений в файл

3. Визуализация
Добавить цветовую индикацию скорости подачи (F-параметры)

Отображать разные типы перемещений разными цветами (G0 - быстрые, G1 - рабочие)

Добавить возможность скрывать/показывать отдельные слои (по Z)

Добавить легенду и цветовую шкалу

4. Анализ G-code
Определение границ модели (min/max координаты)

Расчет объема материала

Определение времени печати

Анализ экстремумов и проблемных зон

5. Экспорт
Экспорт модели в STL/OBJ/PLY форматы

Сохранение настроек вида в файл

Создание анимации облета модели

6. Интерфейс
Добавить строку статуса с информацией о модели

Добавить возможность изменять цвет фона

Добавить лупу для увеличения области

Добавить переключатель режимов рендеринга (wireframe/surface)

7. Работа со слоями
Разделение G-code на слои по высоте Z

Возможность показать только текущий слой

Анимация послойной печати

8. Импорт/Экспорт настроек
Сохранение настроек цвета, толщины, прозрачности

Сохранение позиции камеры

Экспорт текущего вида в изображение с настройками
