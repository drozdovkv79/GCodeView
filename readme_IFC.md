Задача под IFC merg convert view

Программа на Python 3.13 PyQt6 PyVista для просмотра и анализа IFC2X3 очень больших зданий (10 миллионов элементов) и конвертации для просмотра в WEB. Требования:
1. Вся работа программы идет в одном интерфейсе
2. Все логи консоли выводятся в текстовое окно основной формы (на отдельную вкладку)
3. В основном окне должны отражаться: окно с деревом иерархии элементов модели, окно визуализация модели IFC через PyVista, окно просмотра атрибутов модели (группа-атрибут-значение)
4. Добавь на окно кнопки: Конвертировать, Объеденить. По нажатию выводи просто сообщение с наименованием кнопки
5. Есть кнопка выбрать директорию с файлами IFC. При нажатии открывается диалог выбора директории и отображается список файлов IFC из выбранной директории. 
6. Чтений выбранного в списке файла IFC только при нажатии на кнупку "Читать файл" после чего загружается модель IFC и отображается трехмерная визуализация модели, иерархия и атрибуты
7. Фон окна для модели сделай silver. 
8. Шрифт везде Arial 12pt
9. После загрузки IFC в лок выведи подробную статискику по времени загрузки в разрезе: элементов, материалов, цвета и прочих данных IFC.
10. Память IFC: ifcopenshell держит всю модель в RAM. Для файлов >100MB используйте ifcopenshell.open(filepath, lazy=True)
